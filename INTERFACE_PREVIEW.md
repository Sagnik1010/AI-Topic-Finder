## Color Scheme

The interface uses a professional purple gradient theme:

```
Primary Gradient:
╔═══════════════════════╗
║ Darker Purple         │  #667eea at top
║ ↓ ↓ ↓                 │
║ Lighter Purple        │  #764ba2 at bottom
╚═══════════════════════╝

Component Colors:
- Answer Box:    Purple gradient background
- Cards:         Light gray background with blue left border
- Text:          Dark gray on light background
- Buttons:       Purple on white, darker on hover
- Hover States:  Subtle shadow and transform animations
- Timestamps:    Muted gray text with icons
```

## Interactive Elements

### Search Box
- Rounded corners
- Smooth focus animation
- Placeholder text
- Auto-focus on load

### Example Pills
- Bordered buttons
- Hover effect changes color to blue
- Slight lift animation on hover
- Clickable to fill search

### Ask Button
- Large, prominent
- Purple gradient
- Hover: Lifts up slightly
- Active: Returns to normal
- Disabled state: When processing

### Video Cards
- Slide left animation on hover
- Shadow effect on hover
- Flex layout for responsiveness
- Icon support for timestamps

## Animations

```
1. Page Load
   - Title: Fade in from top
   - Search: Slide up from bottom
   - Smooth 0.3s duration

2. Search Response
   - Loading spinner: Continuous rotation
   - Results: Fade in smoothly
   - Cards: Stagger animation

3. Hover Effects
   - Buttons: Lift up 2px
   - Cards: Slide right 5px + shadow
   - Pills: Color change + lift

4. State Changes
   - Loading state: Spinner shows
   - Results appear: Smooth fade in
   - Error: Red highlight
```

## Accessibility Features

- Semantic HTML structure
- ARIA labels on buttons
- Keyboard navigation support
- Proper color contrast (WCAG AA)
- Focus states visible
- Mobile responsive
- Touch-friendly tap targets

## Browser Support

Works on:
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+
- Mobile browsers (iOS Safari, Chrome Mobile)
