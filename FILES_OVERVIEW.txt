🚀 HOW TO RUN (3 STEPS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Terminal 1 - Start AI Engine
   $ ollama serve
   (Wait for: Listening on 127.0.0.1:11434)

2. Terminal 2 - Start Web Server  
   $ python app.py
   (Wait for: Running on http://0.0.0.0:5000)

3. Browser - Open Interface
   http://localhost:5000

WEB INTERFACE
─────────────────────────────────────────────────────────────────────────────
app.py (1.8KB)
  ├─ Flask web server
  ├─ Serves HTML interface
  ├─ Handles /ask API endpoint
  ├─ Manages subprocess calls
  └─ Returns JSON responses

templates/index.html (14KB)
  ├─ Beautiful purple interface
  ├─ Search box & example buttons
  ├─ Loading animations
  ├─ Answer display area
  ├─ Video segment cards
  ├─ Mobile responsive
  └─ Full error handling

AI PROCESSING
─────────────────────────────────────────────────────────────────────────────
process_incoming.py (2.5KB)
  ├─ Receives user question
  ├─ Creates embeddings (Ollama API)
  ├─ Searches for similar videos
  ├─ Builds enhanced prompt
  ├─ Calls LLM (Ollama API)
  ├─ Saves response files
  └─ Returns chunks JSON

embeddings.joblib
  ├─ All video chunks with metadata
  ├─ Pre-computed embeddings
  ├─ Loaded for similarity search
  └─ Contains: title, number, start, end, text, embedding

🎯 WHAT HAPPENS WHEN STUDENT ASKS A QUESTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Input:  Student types: "Where is CSS explained?"
        ↓
Step 1: Question → Vector (AI embedding)
Step 2: Search all video chunks (similarity)  
Step 3: Find 5 most relevant videos
Step 4: Build prompt with video chunks
Step 5: Send to LLM for generation
Step 6: Format answer with timestamps
        ↓
Output: Beautiful answer with video cards showing:
        - Video title
        - Video number
        - Exact timestamp (MM:SS - MM:SS)
        - Relevant transcript text
